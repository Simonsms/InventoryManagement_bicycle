📱 扫码录入商品

自行车库存管理系统 - 移动端入口

https://bike.aipr0.com/scan

📋 使用说明

  1. 手机扫描上方二维码，或直接在手机浏览器输入上方网址
  2. 首次访问需要登录系统（使用管理员或店主账号）
  3. 进入扫码页面后，点击"开始扫码"按钮
  4. 浏览器会请求摄像头权限，点击"允许"
  5. 对准商品条形码或二维码进行扫描
  6. 扫描成功后，填写商品信息（名称、品牌、型号等）
  7. 点击"提交"完成录入
⚠️ 重要提示：
  • 必须使用 HTTPS 访问才能调用摄像头
  • 首次访问可能提示"不安全"，点击"继续访问"即可
  • 建议使用 Chrome、Safari 或 Edge 浏览器
  • 确保手机已连接网络（WiFi 或移动数据）
